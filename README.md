# Managing Complex Change

An interactive reference for the Lippitt-Knoster model of managing complex change.

Every failed change initiative is missing at least one ingredient. The symptom tells you which one. This page helps you find the gap, understand what you're seeing, and know what to do about it.

**Live site:** [tommy2118.github.io/lippitt-knoster](https://tommy2118.github.io/lippitt-knoster/)

## The Model

Five ingredients are required for successful change:

| Missing Ingredient | Symptom |
|---|---|
| Vision | Confusion |
| Skills | Anxiety |
| Incentives | Resistance |
| Resources | Frustration |
| Action Plan | False Starts |
